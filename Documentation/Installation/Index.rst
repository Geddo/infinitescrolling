Installation
============

Target group: **Administrators**

The extension needs to be installed like any other extension of TYPO3 CMS:

#. Visit ExtensionManager

#. Switch over to `Get Extensions`

#. Search for `infinitescrolling`

#. Install extension


**Next step**

Go to Template module and add static template of infinitescrolling. Switch over to constant editor and modify
the jQuery selectors to your needs.
